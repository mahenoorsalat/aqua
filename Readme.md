# Animated Landing Page

## Overview

This project is a sleek and modern landing page featuring smooth animated effects. It is designed to provide a visually appealing introduction to our unique studio.

## Preview

![Landing Page Preview](https://images.pexels.com/photos/5473026/pexels-photo-5473026.jpeg?auto=compress&cs=tinysrgb&dpr=1&w=500)

## Features

- **Responsive Design**: Adapts to various screen sizes for an optimal viewing experience.
- **Animated Elements**: Includes animated bubbles and smooth transitions.
- **Navigation Bar**: Easy navigation with a sign-up button.
- **Social Media Integration**: Quick access to Upwork, Fiverr, LinkedIn, Freelancer, and GitHub profiles.

## Technologies Used

- **HTML5**: For the structure of the web page.
- **CSS3**: For styling and animations.
- **Font Awesome**: For social media icons.
